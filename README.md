# Installation

## Clone

```
git clone https://github.com/jovimoura/ecommerce-next
```

## Access the folder

```
cd <name-folder>
```

## Install dependencies

```
npm i
```

## Start the app

```
npm run dev
```

## Port:

```
http://localhost:3000/
```
